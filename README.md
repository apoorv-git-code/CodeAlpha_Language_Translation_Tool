## 📂 Repository Structure & Projects

Below are the projects included in this repository. Navigate into any specific directory to view its dedicated documentation and setup instructions.

### 1. ⏳ Real-Time Object Detection and Tracking Dashboard 👀
* **Directory:** `Real-time object detection and tracking`
* **Description:** A live Computer Vision pipeline built with Python, OpenCV, and YOLOv8. It features persistent object tracking via ByteTrack and displays a real-time telemetry analytics dashboard over the video feed.
* **Key Features:** Live tracking count, cumulative unique object tracking, automated `.mp4` video recording.

### 2. 🌐 Language Translation Tool 📝
* **Directory:** `CodeAlpha_Language_Translation_Tool` *(Update name if different)*
* **Description:** A utility designed to translate text or documents across multiple languages using natural language processing models.

## 💻 Global Setup & Installation

To get started with any of the projects in this repository, clone the root directory and navigate to the project of your choice.

### 1. Clone the Entire Repository:
```
git clone [https://github.com/apoorv-git-code/CodeAlpha_Artifical_Intelligence.git](https://github.com/apoorv-git-code/CodeAlpha_Artifical_Intelligence.git)
cd CodeAlpha_Artifical_Intelligence
```
### 2. Navigate to a Specific Project: 
For example, to run the Object Detection and Tracking project:
```
cd Real-time\ object\ detection\ and\ tracking
```
### 3. Environment Setup:
It is highly recommended to create a localized virtual environment within the specific project directory you are working in:
```
# Windows
python -m venv venv
venv\Scripts\activate

# macOS / Linux
python3 -m venv venv
source venv/bin/activate
```
## 🛠️ Requirements & Dependencies
Each project contains its own requirements.txt file. Once you have navigated into your project directory and activated your virtual environment, install the specific dependencies using:
```
pip install -r requirements.txt
```
